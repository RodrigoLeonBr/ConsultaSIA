# Pasta de Imagens

## Logo da Empresa

Para adicionar o logo da empresa nos relatórios:

1. Coloque o arquivo do logo nesta pasta com o nome: `logo.png` ou `logo.jpg`
2. Formatos recomendados: PNG (com fundo transparente) ou JPG
3. Tamanho recomendado: 200x80 pixels (ou proporção similar)
4. O logo será automaticamente usado nos cabeçalhos dos relatórios

## Estrutura de Arquivos

- `logo.png` ou `logo.jpg` - Logo principal da empresa
- Outras imagens podem ser adicionadas conforme necessário
